# 🎮 Toy Project - 2D Action Game (SFML)

## 📌 Overview
간단한 2D 액션 게임으로, 플레이어가 몬스터를 처치하며 경험치를 얻고 레벨업하는 구조를 구현한 프로젝트입니다.  
게임 클라이언트 개발에서 중요한 입력 처리, 상태 관리, 전투 시스템, UI를 중심으로 설계되었습니다.

---

## 🛠 Tech Stack
- C++
- SFML 3.0
- Object-Oriented Design

---

## 🎯 주요 기능

### 1. Player 시스템
- 좌우 이동 및 점프 (더블 점프 포함)
- 중력 기반 물리 적용
- 스탯 시스템 (enum + array 기반)
- 경험치 및 레벨 시스템

### 2. 전투 시스템
- 플레이어와 몬스터 충돌 시 데미지 처리
- 몬스터 공격 (쿨타임 기반)
- 플레이어 피격 무적 시간 (Invincibility Frame)

### 3. 몬스터 AI (FSM)
- Idle / Chase / Attack / Dead 상태 기반 동작

### 4. 레벨업 시스템
- 몬스터 처치 시 경험치 획득
- 경험치 충족 시 레벨업
- 3가지 선택지 제공

### 5. 입력 처리
- prev / cur 구조를 활용한 단발 입력 처리

### 6. 몬스터 스폰
- 일정 시간마다 몬스터 생성
- vector 기반 다수 몬스터 관리

### 7. UI 시스템
- Level, HP, EXP 표시
- HP Bar / EXP Bar
- 상태 패널 및 레벨업 UI

---

## 🧠 설계 포인트
- Player: 데이터 및 상태 관리
- Game: 전체 흐름 및 UI 제어
- enum + array 기반 스탯 구조
- FSM 기반 몬스터 AI
- 입력 처리 안정화
